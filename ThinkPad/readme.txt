To find the image associated with the corresponding row
in the CSV file "metadata.csv", search for the last 11
characters in the URL (column B) associated with the row. 
The associated image is titled with the first 11 characters
of the file name in the 'Images' folder. 

For example, consider the URL:
https://purl.stanford.edu/bj993rv7801
                          -----------

The image's file name will be:
bj993rv7801_0001.jp2
-----------
